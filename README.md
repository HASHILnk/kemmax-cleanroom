# KEMMAX Website

Professional clean room engineering company website built with React, Vite, Tailwind CSS, and Framer Motion.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Push this repository to GitHub
2. Import the repo at [vercel.com](https://vercel.com)
3. Vercel auto-detects Vite settings (`build`: `vite build`, `output`: `dist`)
4. Optional: set `VITE_FORM_ENDPOINT` for contact form backend (Web3Forms/Formspree)

## Content Updates

All website copy lives in `src/data/content.ts`. Replace placeholder text and image URLs there.
